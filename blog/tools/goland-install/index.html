InteliJ IDEA安装

网上的InteliJ IDEA破解安装这里便不赘述了,随便百度了一下，推荐一下这个blog 正版安装详细见官网https://www.jetbrains.com/idea/ 安装好IDEA后，settings里面plugin安装go插件。Settings —> pulgins —> Browse repositories —> 搜索go，安装star最多的即可

go安装

下载最新的go版本安装https://golang.org/dl/,win10下载 Installer Windows x86-64。 开始golang吧！
